# AI Fashion Stylist Landing Page

A modern, animated React landing page for an AI Fashion Stylist application featuring:

- **Animated typing headline** with typewriter effect
- **Interactive product hotspots** on model images
- **Smooth scrolling navigation** between sections
- **Responsive design** with mobile-first approach
- **Modern UI components** with custom animations
- **Multiple sections**: About, Services, Pricing, Contact

## Features

- **J.A.R.V.I.S** - Just A Rather Very Intelligent Stylist
- **S.T.A.R.K. Industries** themed design
- **Interactive navigation** with smooth scrolling
- **Product tagging system** with animated reveals
- **Service categories** for different user types
- **Pricing tiers** with waitlist functionality
- **Contact form** with styled inputs

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3001](http://localhost:3001) to view it in the browser.

### Available Scripts

- `npm start` - Runs the app in development mode
- `npm run build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm run eject` - Ejects from Create React App

## Technologies Used

- **React 18** - Frontend framework
- **Tailwind CSS** - Utility-first CSS framework (via CDN)
- **Lucide React** - Icon library
- **Custom CSS animations** - For enhanced user experience

## Sections

1. **Hero Section** - Animated headline with model and product hotspots
2. **About Section** - S.T.A.R.K. Industries system overview
3. **Services Section** - Tabbed interface for different user types
4. **Pricing Section** - Three-tier pricing with waitlist functionality
5. **Contact Section** - Contact form with themed styling

## Customization

The app uses a consistent color scheme:
- Primary: `#4A0404` (Dark Red)
- Accent: `#D90429` (Bright Red)
- Background: `#FDFBF7` (Off White)
- Dark sections: `#050101` (Near Black)

To create a production build:
